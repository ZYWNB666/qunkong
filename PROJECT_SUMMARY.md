# QueenBee 运维管理平台 - 项目总结

## 🎯 项目概述

基于Vue 3 + Element Plus开发的现代化运维管理平台，完全参考了 https://queenbee.gre.garenanow.com/93/execute/script 的设计风格，提供脚本执行、执行历史和Agent管理三大核心功能。

## ✨ 核心功能

### 1. 脚本执行 🚀
- **多语言支持**: Shell脚本和Python脚本
- **实时编辑器**: 带行号的代码编辑器
- **批量执行**: 支持多服务器同时执行
- **参数化**: 支持脚本参数传递
- **超时控制**: 可配置执行超时时间
- **用户权限**: 支持不同执行账号

### 2. 执行历史 📊
- **任务监控**: 实时查看任务执行状态
- **结果展示**: 详细的执行结果和日志
- **筛选搜索**: 多维度筛选和关键词搜索
- **任务管理**: 支持任务重试和停止
- **分页显示**: 高效的数据分页展示

### 3. Agent管理 🖥️
- **状态监控**: 实时监控Agent在线状态
- **系统信息**: 详细的系统资源信息
- **批量操作**: 支持批量重启、更新等操作
- **执行追踪**: 查看Agent的执行历史
- **版本管理**: Agent版本更新控制

## 🛠️ 技术架构

### 前端技术栈
- **Vue 3**: 现代化前端框架
- **Element Plus**: 企业级UI组件库
- **Vue Router 4**: 单页面应用路由
- **Axios**: HTTP请求库
- **Vite**: 快速构建工具

### 后端技术栈
- **Python 3.7+**: 后端开发语言
- **Flask**: 轻量级Web框架
- **WebSocket**: 实时通信
- **SQLite**: 轻量级数据库
